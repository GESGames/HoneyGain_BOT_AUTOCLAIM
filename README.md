2. Go to your Forked Repo **Settings** > **Secrets and variables** > **Actions**.

   - `HG_PASSWORD`: Your Honeygain password.
4. Go to the **Actions** tab and click the green button to enable them.
5. Select `HoneyGain_BOT_AUTOCLAIM` on the left and click **Run workflow** to test it!
